Sites offering software using the JSAPI
  • JBrix - Offers a Swing-based tool (Xybrix) for making applications that work with XML, defining them using XML! It also offers "SayWhat" - which is a JSAPI-enabled Xybrix plugin that allows all applications developed with the Xybrix framework to be operated by voice. Also has other cool tools and APIs. Note: you will need to download the "IBM" version of "SayWhat" rather than the "CloudGarden" version - or recompile the "SayWhat" code - because of the change (as of version 1.2.1) - of the JSAPI field values to match those of the IBM implementation.
  • AliceTalker - inspired by the now-defunct AliceSpeak project by John Janacek, AliceTalker is a free fully Java-based Swing client for the Alicebot project.
  • SpeechPad - by Vinod Kumar - a speech-enabled Notepad using Swing (Version 1.1 now released).
