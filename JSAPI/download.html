Download Cloudgarden's TalkingJava SDK (with JSAPI 1.0 implementation)

The TalkingJava SDK can be downloaded as either a zip file or a java installer.
Both contain the same runtime libraries, examples and documentation.
  • The TalkingJava SDK is FREE for NON-COMMERCIAL use only.
  • Use of the TalkingJava SDK or runtime libraries by a company, corporation or institution requires purchase of one or more Professional Licenses.

Download java installer
Note: If you use the java installer and have already installed version 1.6.3 then you should first un-install version 1.6.3 (by running the installer and choosing the "Uninstall SDK" button) and then install 1.7.1.
