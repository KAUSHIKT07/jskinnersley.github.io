Validate License for Cloudgarden's JSAPI 1.0 implementation

If you purchased a license for any version of Cloudgarden's JSAPI implementation, and are unable to connect to the internet from the machine on which you wish to validate your license, enter in the registration code from the email you received after you purchased your license (or by using this web page), along with the email address you used to purchase the license and the machine code which you can obtain by running the LicenseManager tool that comes as part of TalkingJava 1.6.0.

The following screen will give you your license, which you will need to save to a file and transfer to the machine on which you are running the TalkingJava SDK.
