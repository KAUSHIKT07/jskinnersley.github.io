Purchasing Licences

The Personal and Professional licenses may both be purchased on-line (by PayPal) or by check.

Click on the "Download" link, agree to the license, then choose one of the "Buy Now" links, which will describe the options.

Pricing and Licence Terms

  • Personal License - for personal, academic, non-profit or non-corporate.
    • the price is $16 for a single-user license.
    • The Personal License does not allow the holder to re-distribute the non-trial runtime libraries with an application, but the holder may distribute the time-limited evaluation runtimes.

     
  • Professional License - for companies developing with the implementation (even in exploratory phases of product development) or using the product commercially.
    • The price is $420 for a single-developer license, which covers:
    • Installation for the purposes of development - one license must be purchased for each developer working with the implementation.
    • Redistribution of up to 1000 copies of the runtime libraries bundled with any product.
    • If the implementation is to be redistributed "silently" as part of an applet product (and will not be used except as part of that applet in a web browser) then purchase of two professional licences allows unlimited distribution of the applet via a web server.
    • Installation on up to 50 servers (eg. web-servers or machines handling network requests) when used to process speech as part of a server-side application.


    So, for example, with two Professional licences you could

    • distribute 2000 copies of the runtimes of a (non-applet) product
    • install it on 100 web servers
    • serve it without limit from a web server with an applet and
    • have two developers develop with it.

     
  • Academic Site License - for universities wishing to have a large number of students program with the JSAPI as part of a course:
    • the price is $500 for a single license.
    • the license includes all the terms of the professional license agreement but in addition allows the JSAPI implementation to be used by up to 100 students for development as part of their course work, but not for their personal use outside of course work.
